# JAVA基础

## 线程池

new ThreadPoolExecutor(10,10,200L, TimeUnit.MILLISECONDS
,new LinkedBlockingQueue<Runnable>(10),
Executors.defaultThreadFactory(),
new ThreadPoolExecutor.AbortPolicy()   
参数：核心线程数，最大线程数，非核心线程存活时间，存活时间单位，阻塞等待队列，线程工厂，丢弃策略  
执行逻辑：核心线程空闲直接执行，核心满了优先放到阻塞队列等待，阻塞队列满了创建新线程直到达到最大线程数，达到最大线程且阻塞队列满了执行丢弃策略  
ThreadPoolExecutor自带丢弃类型：  
DiscardOldestPolicy-随机丢弃一个老的等待任务  
DiscardPolicy-直接丢弃  
AbortPolicy-抛异常  
CallerRunsPolicy-由提交任务的调用者线程直接执行任务
