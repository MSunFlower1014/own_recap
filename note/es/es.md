# es

## 基础

Index (索引)：类似于关系型数据库的 Database（在 7.x 之后也对应 Table 的概念，因为移除了 Type）。  
Document (文档)：索引中存储的一条数据，JSON 格式，类似于 Row。  
Field (字段)：文档中的属性，类似于 Column。  
Mapping (映射)：定义字段的类型、分词器等，类似于 Schema。  
Shard (分片)：索引的物理切分，实现分布式存储。
Replica (副本)：分片的备份，实现高可用和负载均衡。

## 倒排索引

将字段分词解析后保存映射关系，查询直接查询关键字对应的文档id，然后查找文档全部信息

## 深度分页

普通分页from+size，但大数据量时及其消耗内存和cpu  
scroll快照查询，用于大数据导出，根据返回的id获取指定size数据

## 性能优化

不需要搜索的字段不进行索引映射（index：false）
